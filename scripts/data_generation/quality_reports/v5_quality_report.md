## 四、已知局限 (v6 迭代建议)

1. **部分题目 query 冗余**: 少数从 DeepSeek 提取的条款作为 query 时过长，建议后续优化模板截断逻辑
2. **rubric_clauses 覆盖率**: L1 rubric 通过正则构建，部分纯文字答案引用标准条款的格式不标准，导致提取不完整
3. **非电气关键词补充**: 需添加 `排水|生活污水|污水|采暖|通风` 到过滤列表
4. **L2/L3 问题**: v4 中 L2/L3 存在 rubric_clauses 不足（70题）、非电气内容（L2:7题/L3:25题含环保）等问题，建议 v6 修复
